*관리자 메인 페이지 (메인페이지에서 로그인 페이지로 리다이렉션 할거임-> 로그인을 안했으니까)
1. 로그인 화면 띄우기 매핑 페이지-> get
2. 로그인 버튼 ->Post
3. 로그인 버튼 클릭 후 메인페이지로 이동 매핑 -> get
4. 로그아웃 버튼 -> post
5. 로그아웃 버튼 클릭 후 메인페이지로 이동 매핑 ->  get

*상품등록
1. 상품등록 메뉴 클릭했을 때 페이지 이동 매핑 -> get
2. 상품등록 버튼 -> Post
3. 상품등록 버튼 클릭 후 상품목록 페이지로 이동하기 매핑 -> Post

*상품목록
1. 상품목록 메뉴 클릭했을 때 목록 페이지 이동 매핑->get
2. 상품목록 목록페이지에서 제목 클릭했을 때 상세페이지로 이동->get
3. 상품목록 "상세페이지에서 수정버튼" 눌렀을 때 등록했던 페이지로 이동(당연히 저장되어있어야함) ->get
4. 상품목록 등록했던 페이지에서 내용 수정 후 등록하기 버튼 -> post
5. 상품목록 등록했던 페이지에서 내용 수정 후 등록하기 버튼 누른 후 목록페이지로 이동 매핑 -> get
6. 상품목록 상세페이지에서 삭제 버튼 ->post
7. 상품목록 상세페이지에서 삭제 버튼 클릭 후(삭제ok) 상품목록 페이지로 이동 매핑 ->get
*상품목록 목록페이지 체크박스
6. 상품목록 목록 페이지에서 "체크박스로 선택 후 수정버튼" 바로 눌렀을 때 등록했던 페이지로 이동 매핑 ->get (3번과 같음)
7. 상품목록 목록 페이지에서 "체크박스로 선택 후 삭제버튼" 바로 눌렀을 때 삭제됨 매핑 ->post
